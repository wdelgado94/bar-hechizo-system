# Sistema de Gestión - Bar Hechizo

## Descripción

Sistema web para la administración integral del Bar Hechizo.

## Funcionalidades

- Gestión de mesas
- Gestión de ventas
- Gestión de inventario
- Apertura y cierre de caja
- Gestión de gastos
- Reportes administrativos

## Tecnologías

### Backend
- Node.js
- Express

### Frontend
- React
- Next.js

### Base de Datos
- PostgreSQL

## Estado

Proyecto en fase de análisis y diseño.
