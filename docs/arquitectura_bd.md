# Sistema Bar Hechizo

## Base de Datos

### Tablas

#### mesas
Control de mesas del establecimiento.

#### usuarios
Administradores y meseros.

#### categorias
Clasificación de productos.

#### productos
Inventario principal del bar.

#### presentaciones
Formas de venta de los productos.

Ejemplo:
- Botella
- Media botella
- Shot sencillo
- Shot doble

#### ventas
Cabecera de las ventas.

#### detalle_ventas
Detalle de productos vendidos.

#### cajas
Control de apertura y cierre de caja.

#### movimientos_inventario
Entradas, salidas, mermas y ajustes.

#### gastos
Registro de gastos operativos.
